End-to-End Project Management (Сервис «Семейный бюджет»)

Роль: Project Manager (Fintech)

Контекст: Разработка нового функционала для банковского приложения, позволяющего семьям совместно управлять финансами в рамках жесткого дедлайна (4 месяца) и строгих регуляторных требований.

Discovery & Planning:

Провел сбор требований через интервью со стейкхолдерами (бизнес, техлид) и анализ документации.

Сформировал MVP-объем из 12 ключевых User Stories (категория Must-have), сфокусировавшись на вовлеченности пользователей и кросс-продажах банковских продуктов.

Определил архитектурные инварианты: SLA 99,5%, шифрование по ГОСТ, поддержка iOS/Android.

Methodology & Governance:

Обосновал и внедрил гибридный Agile-подход (Scrum + Kanban) для обеспечения итеративной поставки ценности.

Организовал цикл встреч: Daily Standups (15 мин) для снятия блокеров, Demo для сбора фидбэка от заказчика и Retrospectives для улучшения процессов команды.

Documentation:

Организовал ведение Product Backlog в Yandex Tracker.

Создал базу знаний в Yandex Wiki для фиксации ключевых решений и снижения рисков недопонимания при ограниченных ресурсах на анализ (≤3 чел-нед).

Budgeting & Resource Management:

Рассчитал стоимость разработки проекта на основе трудозатрат кросс-функциональной команды (15+ ролей: от архитекторов до Devops-инженеров).

Работал с детализированными шаблонами биллинга, учитывая рыночные ставки специалистов и количество итераций (дизайн-макеты, верстка, логика API).
